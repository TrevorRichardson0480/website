Swamp Hacks 2020

  • This was my first Hackathon and I was very overwhelmed (but in a good way). I tried to make it to as many workshops as possible, which was impossible because I also had a symphonic performance that weekend too! I enjoyed meeting many people, companies and learning so much about the industry. My group's project was a very simple game using an Arduino and other electronics. We called it Cody (get it, like code - y). I wasn't expecting to do anything extravagant since this was my first Hackathon and I didn't have much time, but at least I learned how to code for Arduino. I'm a freshman so I really still don't know a lot, but SwampHacks was great! And the swag... there was so much swag.
