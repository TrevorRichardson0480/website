Mood Teller

    Previous Next
  • Knowing about the similarities between C# and Java, I created this program to just get a good understanding of C#. I also watch an hour-long video on just a quick introduction to C#. The similarity is very obvious, and I found this mini-project to be pretty fun. What the "Mood Teller" will do is use an algorithm to tell whether the user is in a good mood or a bad mood. Using an array of good-mood words, an array of bad-mood words, and an array of opposite words (such as "not"), the algorithm will take in a sentence from the user after asking the user about their mood. The algorithm will then search for good-mood words, bad-mood words, and opposite words in the sentence. Then it will generate a "mood score" and use that to determine the user's mood. The algorithm will also be able to tell if the sentence may have been a little confusing. In the end, the algorithm will tell the user if it thinks they are in a good mood, a bad mood, if it can't tell, and if the result may be a little off. - 4/6/2020 - C#

Economics Program

  • I wanted to play around with reading from and writing to text files. So this java program will allow you to keep track of your monthly expenses and income. Data will be saved to and called from text files. Two text files are used per set of data (the program will call it a "month", assuming the user is keeping track of their expenses and income by month.) The first text file keeps track of daily changes (ie. the difference between the income and expenses for that day). This file is called the month file, and will also hold the statistics. The second file will keep track of every income and expense, regardless of the day. This is called the log file, and any data that has been updated or "written over" will be marked as "VOID" on the log file. The user will be able to add various types of income and expenses, as well as set to a different month, view the files within the program, view and update statistics, set a monthly goal, and export the files to the desktop, documents, or another desired directory. This is by far my biggest project yet! - 4/5/2020 - Java
